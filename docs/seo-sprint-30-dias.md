# Sprint SEO de 30 días para arrendamiento puro

Objetivo: subir posiciones para búsquedas comerciales como `software de arrendamiento puro`, `software para arrendadoras`, `software para contratos de arrendamiento puro`, `consulta de multas vehiculares para arrendadoras` y `control de activos para arrendamiento puro`.

## Semana 1: base técnica y contenido

- Reforzar la página principal de software de arrendamiento puro.
- Agregar schema de software y FAQ.
- Publicar páginas satélite:
  - `software-contratos-arrendamiento-puro`
  - `consulta-multas-vehiculares-arrendadoras`
  - `control-activos-arrendamiento-puro`
- Actualizar sitemap y solicitar indexación en Search Console.

## Semana 2: autoridad externa

- Publicar 2 posts en LinkedIn con capturas demo.
- Agregar enlaces a `janco.tech` desde perfiles y README de GitHub.
- Buscar 2 enlaces externos reales: cliente, aliado, directorio o publicación.

## Semana 3: ajuste con Search Console

- Revisar consultas con impresiones.
- Mejorar títulos y primeros párrafos de páginas que ya estén apareciendo.
- Identificar keywords con impresiones sin clics.
- Agregar una sección nueva si Google muestra una intención clara.

## Semana 4: conversión y prueba comercial

- Revisar qué páginas reciben clics y qué formularios generan leads.
- Ajustar CTA de WhatsApp/demo en páginas con más impresiones.
- Preparar una landing corta para campaña de Google Ads si se quiere acelerar primera página.
- Decidir si conviene crear una página comparativa: `software de arrendamiento puro vs sistema financiero adaptado`.

## Revisión semanal

Métricas mínimas:

- Impresiones por consulta.
- Posición promedio por página.
- Clics orgánicos.
- Leads por formulario o WhatsApp.
- Páginas indexadas en Search Console.
